# Olist 电商平台 GMV 异动分析报告

## 一、分析目标

在对 Olist 电商平台订单数据进行月度趋势追踪时，发现 **2017 年 11 月至 12 月 GMV 环比下降 26%**，为全年最显著的下滑阶段。
 本项目旨在：

- 识别 GMV 异动的主要来源（时间与品类维度）；
- 拆解 GMV 的变化驱动（订单量 vs 客单价）；
- 为后续履约与地理层面分析提供线索。

## 二、数据概况

- **数据来源：** Olist 官方公开数据集
- **使用表：**
  - `orders_table`（订单表）：订单 ID、下单时间、发货时间、状态等；
  - `order_item_table`（订单明细表）：订单与商品关联、单价、运费；
  - `products_table`（产品表）：产品 ID、品类信息。
- **核心字段：**
  `order_id`, `product_id`, `order_purchase_timestamp`, `order_delivered_customer_date`, `price`, `freight_value`
- **时间范围：** 2017 年 1 月至 2018 年 8 月
- **GMV 计算方式：** `GMV = price + freight_value`



## 三、阶段一：月度 GMV 趋势与异常识别

GMV 总体走势

| 月份        | 订单数 | 环比       | 客单价 | 环比  | 月GMV      | 环比       |
| ----------- | ------ | ---------- | ------ | ----- | ---------- | ---------- |
| 2017-10     | 4478   | +7.9%      | 157.3  | -1.0% | 704k       | +6.8%      |
| **2017-11** | 7289   | **+62.8%** | 150.1  | -4.6% | **1,093k** | **+55.3%** |
| **2017-12** | 5513   | **-24.4%** | 146.3  | -2.5% | **807k**   | **-26.3%** |

**发现**

- GMV 于 2017 年 11 月出现异常峰值，12 月显著下滑（环比 -26%）。
- 通过分解 GMV = 订单量 × 客单价，可见：
  - 主要由订单量下降（-24%）导致；
  - 客单价略有下滑（-2.5%），贡献有限。

 **初步假设**
 GMV 下滑可能由：

- 季节性促销（11 月黑五）导致需求提前释放；
- 品类结构变化（头部品类退潮）；
- 供应链或履约因素导致转化下降。



## 四、阶段二：品类维度拆解

####  品类集中度

- 共识别出 73 个产品品类，呈现明显的长尾结构；
- 采用帕累托（80-15-5）法则聚类：
  - 前 80% GMV：核心类目；
  - 15%：`mid_others`；
  - 5%：`long_tail`；
  - 未知品类统一归入 `unknown`。

####  11–12 月品类 GMV 表现

> 下表展示主要类目的 GMV 环比变化及对整体下滑的贡献率👇

| 品类                               | GMV变化率 | 贡献率 | 说明             |
| ---------------------------------- | --------- | ------ | ---------------- |
| cama_mesa_banho（家居床品）        | -43%      | 15%    | 季节性回落显著   |
| moveis_decoracao（家居装饰）       | -51%      | 13%    | 家具消费周期性强 |
| informatica_acessorios（数码配件） | -48%      | 12%    | 黑五透支需求     |
| relogios_presentes（礼品钟表）     | -27%      | 9%     | 节日礼品需求减弱 |
| beleza_saude（美容健康）           | -24%      | 7%     | 消费优先级下调   |
| ferramentas_jardim（园艺工具）     | -38%      | 7%     | 季节性需求下滑   |

**结论：**
 11–12 月 GMV 下滑由多核心类目共同拖累，增长类目体量有限，难以对冲整体疲软。



## 五、阶段三：GMV 拆解分析（一级指标）

| 品类                   | GMV变化率 | 订单量变化率 | 客单价变化率 |
| ---------------------- | --------- | ------------ | ------------ |
| beleza_saude           | -24%      | -18%         | -8%          |
| cama_mesa_banho        | -43%      | -41%         | -4%          |
| ferramentas_jardim     | -38%      | -43%         | +10%         |
| informatica_acessorios | -48%      | -42%         | -10%         |
| moveis_decoracao       | -51%      | -43%         | -14%         |
| relogios_presentes     | -27%      | -21%         | -7%          |

**拆解发现：**

- 主要下滑由订单量减少驱动；
- “园艺工具”虽 GMV 下滑，但客单价上升（+10%），或因高价品销售占比上升；
- 整体需求端活跃度下降，定价变化影响较小。

 **结论：**
 GMV 下滑的核心原因为销量萎缩，而非价格变化。可进一步验证是否存在季节性促销影响或库存周期错位。



### 六、阶段四：分析结论与启示

####  结论总结

1. **问题发现：**
   11–12 月 GMV 环比下降 26%，为全年最明显波动。
2. **关键驱动：**
   主要由订单量减少推动，价格因素次要。
3. **核心责任类目：**
   七大类目贡献 GMV 下滑超 60%，头部品类失速是主因。
4. **可能成因：**
   - 黑五促销需求前置；
   - 季节性消费回落；
   - 高价值品类周期性下降。
